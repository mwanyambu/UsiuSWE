A Book Store Web App where users can:
✅ View a list of books (stored in an XML file).
✅ Filter/search books using JavaScript.
✅ Add books to a shopping cart.
✅ Store cart data in local storage.
This project reinforces frontend web development skills, including:
✔ JavaScript DOM manipulation
✔ XML data handling
✔ AJAX for loading XML dynamically
✔ Local Storage for data persistence
