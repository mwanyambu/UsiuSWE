# LearnHub - Interactive E-Learning Platform

## Project Overview
LearnHub is a responsive and accessible e-learning platform that showcases web development skills including HTML5, CSS3, JavaScript, and XML data handling. The website includes multiple interlinked pages, interactive elements, and follows web standards for accessibility.

## Features

### HTML Structure
- Multiple interlinked pages (Home, Courses, About)
- Semantic HTML5 elements (header, nav, section, article, footer)
- Proper use of headings, lists, and tables
- Embedded images with appropriate alt text

### CSS Implementation
- External CSS file for styling
- Responsive design using media queries
- CSS Box Model with appropriate margins, padding, and borders
- Flexbox and Grid for layout design

### JavaScript Interactive Elements
1. Light/Dark Mode Toggle
   - Users can switch between light and dark themes
   - Theme preference is saved in localStorage

2. Form Validation
   - Newsletter subscription form validation
   - Contact form validation with appropriate error messages

3. Dynamic Content Loading
   - Course data is loaded from XML file
   - Search and filtering functionality for courses

### XML Data Handling
- Well-formed XML file storing course data
- Dynamic rendering of course information using JavaScript and DOM manipulation
- XML data used to populate course listings and detailed views

### Accessibility Features
- ARIA attributes for better screen reader support
- Proper color contrast between text and background
- Keyboard navigation support
- Responsive design for all device sizes

## File Structure
- index.html - Homepage with featured courses
- courses.html - Course catalog with search and filtering
- about.html - About page with contact form
- css/styles.css - Main stylesheet
- js/script.js - Main JavaScript functionality
- js/courses.js - Course-specific JavaScript
- js/contact.js - Contact form validation
- data/courses.xml - XML file containing course data
- images/ - Directory containing SVG images for courses and team

## How to Run
1. Unzip the project files to a directory
2. Open index.html in a web browser

## Validation
All HTML and CSS files have been validated using the W3C Validator to ensure they meet web standards.
